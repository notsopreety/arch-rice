# arch-rice

Personal Arch Linux + Hyprland rice using Quickshell with Material You dynamic theming.

## Window Manager

| Component | Tool |
|-----------|------|
| **Compositor** | [Hyprland](https://hyprland.org/) (Lua-based config via `hyprland.lua`) |
| **Idle Daemon** | `hypridle` |
| **Wallpaper** | `awww` daemon + `awww` client with transitions |
| **Screen Locker** | Custom Quickshell lock screen (PAM auth, wlroots session lock) |
| **Clipboard** | `cliphist` + `wl-paste` |
| **Screenshots** | `grim` / `slurp` integrated via Quickshell |
| **Layout** | Niri-style column navigation (dwindle / scrolling layout) |

## Desktop Shell (Quickshell)

Full [Quickshell](https://quickshell.dev/) QML-based desktop environment with:

| Feature | Description |
|---------|-------------|
| **Bar** | Floating pill-style top bar (workspaces, system monitor, media player, clock, weather, connectivity, audio, battery, system tray) |
| **App Launcher** | Full-screen overlay with search + category grid |
| **Control Center** | Quick settings toggles (15+ toggles) |
| **Notification Center** | History grouped by app with DND mode |
| **OSD** | Android 16-style on-screen displays (volume, brightness, media, USB, etc.) |
| **Lock Screen** | Wallpaper background, media card, weather, CAVA visualizer, PAM auth |
| **Power Menu** | Lock, sleep, reload, restart, logout, power off |
| **Workspace Overview** | Grid with live window previews |
| **System Monitor** | Multi-page (CPU, GPU, memory, disk, network, battery, processes) |
| **Wallpaper Picker** | Carousel with thumbnail caching |
| **Sidebar** | AI Chat (Gemini 2.5 Flash), booru image browser |
| **Clipboard Manager** | History overlay |
| **Emoji Board** | Emoji picker with categories |
| **Keybinds Cheatsheet** | On-screen keybind reference |
| **Notepad / Calculator** | Floating utility overlays |
| **Google Lens Overlay** | Screen snip + search |
| **Snip Tool** | Region screenshot |
| **USB Hotplug OSD** | Auto-detection of external storage |
| **Battery Monitor** | Audio alerts on connect/disconnect/low/full |
| **Media Player** | MPRIS integration in bar |
| **Alarms / Timer / Stopwatch** | Built-in time utilities |

## Color System

| Component | Details |
|-----------|---------|
| **Generator** | [matugen](https://github.com/InioX/matugen) - Material You color extraction from wallpaper |
| **Scheme** | Material Design 3 (`scheme-content`) |
| **Outputs** | Quickshell (`colors.json`), Hyprland (`colors.lua`), Kitty (`colors.conf`), JS (`colors.js`) |
| **Dynamic** | Colors auto-regenerate on wallpaper change; Quickshell watches `colors.json` reactively |
| **Dark Mode** | Enabled by default |

## Directory Structure

```
arch-rice/
├── cava/           # Audio visualizer (config + GLSL shaders + themes)
├── fastfetch/      # System fetch config
├── fish/           # Fish shell config
├── hypr/           # Hyprland Lua config (10 modules)
│   └── lua.d/      # programs, monitors, autostart, env, input, keybinds, look_and_feel, rules, colors, permissions
├── kitty/          # Terminal emulator config
├── matugen/        # Material You color pipeline (config + 5 templates)
│   └── templates/  # colors.json, colors.lua, colors-kitty.conf, colors.js, quickshell_colors.json
├── quickshell/     # Main desktop shell (QML)
│   ├── assets/     # Fonts, icons, sounds, weather SVGs, logos
│   ├── components/ # Reusable QML components (bar modules, sliders, clocks, etc.)
│   ├── core/       # Singletons (Config, Appearance, GlobalStates, Directories, Images)
│   ├── scripts/    # Python/Bash helper scripts
│   ├── services/   # 50+ QML service objects
│   ├── shaders/    # Qt shader binaries
│   ├── theme/      # Material You 3 theme singleton
│   └── widgets/    # 25+ overlay windows (bar, lockscreen, launcher, etc.)
└── scripts/        # Custom shell scripts (wall.sh, battery-daemon.sh)
```
